数据库缓存

2019/10/4 posted in  高并发

缓存数据是为了很少或不访问数据库服务器，高并发下，最大程度降低对数据服务器压力。

启用Mysql查询缓存

query_cache_type
0: 不使用查询缓存
1：始终使用查询缓存
2：按需使用查询缓存

// 不使用
select SQL_NO_CACHE * from my_table;
// 使用
select SQL_CACHE * from my_table;
// 缓存大小
SET GLOBAL query_cache_size = 133333;
// 查看缓存命中次数
SHOW STATUS LIKE 'Qcache_hits'

FLUSH QUERY CACHE;
RESET QUERY CACHE;
FLUSH TABLES;
